# TaskManager

A simple and responsive task management application built with React.

## Features

- Create tasks
- Complete tasks
- Delete tasks
- Search tasks
- Local persistence using Local Storage
- Responsive UI

## Built With

- React
- JavaScript
- CSS
- Local Storage
